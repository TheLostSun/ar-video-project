AR Video Placement Project

Instructions:
1. Place your video file in this same folder.
2. Name it exactly: video.mp4
3. Host this folder using HTTPS (GitHub Pages works).
4. Open on a WebXR compatible mobile browser (Chrome Android).

Flow:
- Tap START AR
- Move device to detect surface (reticle appears)
- Tap to place video
- Use Reset or Restart buttons as needed
